# Voice - Activated Personal Assistant
An interactive, Python-based personal assistant that performs tasks via voice commands. This project performs a variety of key functions such as checking the weather of a particular location, providing the user with the current time and also the latest headlines accurately.

Modules Used in this Project are as follows:

Speech Recognition: Converts the audio captured from the device microphone into text.

Sound Device: Records raw audio data from device microphone.

pyttsx3: A Text-to-Speech (TTS) library that allows the assistant to talk back.

Parses RSS feeds (like BBC News) so the assistant can read headlines.


Key Takeaways

Weather: Fetches real-time weather reports for any city via `wttr.in`.

Global News: Reads the top three headlines from BBC News using RSS feeds.
